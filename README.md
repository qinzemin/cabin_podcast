# AI资讯助手 · 车机前端交互原型

> 智能座舱车机系统「AI资讯助手」前端原型，适配 **1260 × 600** 车机中控屏分辨率。

## 项目结构

```
car-ai-news-assistant/
├── index.html          # 主页面入口
├── styles/
│   ├── reset.css       # 基础重置样式
│   └── main.css        # 全部 UI 样式
├── scripts/
│   └── app.js          # 交互逻辑 / 页面路由 / 数据
└── README.md
```

## 页面与交互说明

| 页面 | 入口 | 说明 |
|------|------|------|
| 主对话框 | 默认首页 | 语音状态区 + 音频卡片列表 + 联想词快捷栏 |
| 偏好设置 | 点击「偏好设置」Chip | 8分类多选，确认/取消，含高阶入口 |
| 高阶资讯订阅 | 偏好设置页底部入口 | 个性化提示语输入 + 付费状态提示 |
| 高阶付费包弹窗 | 点击「立即开通」 | 两档套餐选择 + 支付方式切换 |

## 快速启动

直接用浏览器打开 `index.html` 即可预览（无需构建）：

```bash
open index.html
# 或
npx serve .
```

建议使用 Chrome，并在 DevTools → Device Toolbar 中设置分辨率为 **1260 × 600**，
或直接设置浏览器窗口尺寸以匹配车机屏幕。

## 设计规范摘要

- 目标分辨率：1260 × 600
- 对话框尺寸：宽 364px，最大高 448px，距顶 ≥ 64px，水平居中
- 触控区域：所有可交互元素最小点击范围 44 × 44px
- 色彩体系：深色底 `#0b0f1a`，主题蓝 `#4a90e2`，高阶金 `#d4af37`
